SKSDEV Theme
============

Tags: responsive-layout, white, featured-images, Custom Header, full-width-template, post-formats

A responsive theme with a clean look. A fixed header, and CSS3 transitions effects.This is free mobile template , HTML5 responsive layout with parallax scrolling. It features simple fading image slider and gallery section integrates responsive lightbox.

* HTML5 responsive layout with parallax scrolling
* Full-bleed flex slider
* Featured Image support
* Full-width page template
* Responsive layout
* Custom Header
* CSS3 transition effects
* The GPL license in license.txt. :) Use it to make something cool.


Install Steps:
--------------

SKSDEV includes an full-width slider, which requires the to easily set up and configuration.

Install Required Plugins: Sksdev Toolkit 

Activate plugin for more features and sections show in SKS Theme.

Contact Form show in frontend.

Show in dashboard menu
• Portfolio
• Team 
• Skills

 

Please Goto-> Dashboard-> Appearance -> Theme Options  "Settings" tab show for theme options page.
1. Header tab to show fav icon,  site logo, site title, unlimited theme color layout options.
2. Social icon link show.
3. Slider set your slider image and title button title and links.
4. Services icon title and description.
5. Portfolio seperate setion avilable Goto-> Dashboard-> Portfolio.
6. Team seperate setion avilable Goto-> Dashboard-> Team.
7. Skills seperate setion avilable Goto-> Dashboard-> Skills
8. Contact us section you can chnage you want to show like google map, content, phone, email, address etc.
